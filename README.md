<h1>SendEase (Automated WhatsApp Message Sender)</h1>
<h2>Project Description:</h2>
<p>Developed a Python-based application to schedule and send WhatsApp messages automatically using the Twilio API. The project allows users to input recipient details, compose a message, and specify a future date and time for delivery.</p>
<h2>Key Features:</h2>
    <ul>
        <li>Integration with Twilio API to send WhatsApp messages.</li>
        <li>Scheduling mechanism using Python's <code>datetime</code> and <code>time</code> modules.</li>
        <li>Secure management of API credentials through a custom <code>secure</code> module.</li>
        <li>Input validation to ensure accurate scheduling and error handling.</li>
        <li>Modular and reusable code structure for scalability.</li>
    </ul>
